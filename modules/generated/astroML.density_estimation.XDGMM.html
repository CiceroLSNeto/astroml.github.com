11.2.2.1. astroML.density_estimation.XDGMM¶

class astroML.density_estimation.XDGMM(n_components, n_iter=100, tol=1e-05, verbose=False)¶

Extreme Deconvolution

Fit an extreme deconvolution (XD) model to the data

Parameters :

n_components: integer :

number of gaussian components to fit to the data

n_iter: integer (optional) :

number of EM iterations to perform (default=100)

tol: float (optional) :

stopping criterion for EM iterations (default=1E-5)

Notes

This implementation follows Bovy et al. arXiv 0905.2979

Methods

fit(X, Xerr[, R]) Fit the XD model to data
logL(X, Xerr) Compute the log-likelihood of data given the model
logprob_a(X, Xerr) Evaluate the probability for a set of points
sample([size])
