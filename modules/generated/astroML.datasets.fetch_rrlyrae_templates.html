10.4.21. astroML.datasets.fetch_rrlyrae_templates¶

astroML.datasets.fetch_rrlyrae_templates(data_home=None, download_if_missing=True)¶

Loader for RR-Lyrae template data

These are the light-curve templates from Sesar et al 2010, ApJ 708:717

Parameters :

data_home : optional, default=None

Specify another download and cache folder for the datasets. By default all scikit learn data is stored in ‘~/astroML_data’ subfolders.

download_if_missing : optional, default=True

If False, raise a IOError if the data is not locally available instead of trying to download the data from the source site.
Returns :

data : numpy record array

record array containing the templates
